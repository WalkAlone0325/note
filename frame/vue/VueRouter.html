# 1. VueRouter 解决的问题

  • 监听 URL 的变化，并在变化前后执行相应的逻辑
  • 不同的 URL 对应不同的组件
  • 提供多种方式改变 URL 的 API（URL 的改变不能导致浏览器刷新）

# 2. 使用方式

  • 提供一个路由配置表，不同 URL 对应不同组件的配置
  • 初始化路由实例 new VueRouter()
  • 挂载到 Vue 实例上
  • 提供一个路由占位，用来挂载 URL 匹配到的组件
obj: {
  arr: [{ bool: true }, { bool: false }];
}
